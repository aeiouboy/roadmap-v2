Step 1: Export from Old Roadmap

1. Open your old product-roadmap.html file in your browser

2. Press F12 to open Developer Console

3. Copy and paste this code into the console:

console.log(JSON.stringify(localStorage.getItem('roadmapState')))

4. Press Enter - it will show your data

5. Right-click the output and select "Copy string contents"

Step 2: Paste Data Here


Alternative: Direct Import Script

If you have the old roadmap open in another tab, you can also:

1. Go to the old product-roadmap.html tab

2. Press F12, paste this in the console:

// Copy this data const data = localStorage.getItem('roadmapState'); prompt('Copy this data:', data);

3. It will show a popup with your data - copy it

4. Come back here and paste it in Step 2
